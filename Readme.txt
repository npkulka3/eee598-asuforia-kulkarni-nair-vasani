ASUForia project

Output :
In this app, the camera preview is seen in the on the screen. Each frame is captured and processed to estimate the pose.
To know the final output image, a debug point is to be placed at the return of the onImageAvailabe() in ASUForia.java file.

App Screen :
The name of the app is 'once_more'. The OpenCv button on the screen shows the version of OpenCv used by the app.
The other button is bogus i.e. clicking it has no effect.

Build:
OpenCv 3.3.1

File Required :
The reference image is to be placed in the res folder , and the name of the file should be vuforia.jpg
